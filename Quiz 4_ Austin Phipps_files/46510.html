Consider the brute-force string-matching algorithm discussed in class. Provide an example of a text consisting of 7 characters and a pattern consisting of 3 characters that represents the worst case for the brute-force string-matching algorithm. In your example, the pattern should occur only once in the text.
Your Answer:

Pattern: AAB

Search Worst Case: AAAAAAB

Example Text: AAAAAAB

Example Pattern: AAB

In the brute-force algorithm:

    - The pattern starts matching at position 0 of the text (AAA vs AAB) → mismatch at the 3rd character.

    - Then it shifts by one position and tries again (AAA vs AAB), mismatch again. This continues for each starting position until the last possible alignment.

    - Finally, when the pattern aligns starting at position 4 (AAB vs AAB), it matches.

Because each of the first several shifts compares all characters before failing, this example produces the maximum number of comparisons.

This is part of the bubble sort process to put the biggest number 30 to the index 4. The values of the array values[ ] will be 10, 25, 15, 20, 30 after the process.

=================================================================

int main() {
    int values[5] = {25, 10, 30, 15, 20};
    int size = 5;
    int tmp;
    for (int j = 0; j < size - 1; j++) {
        if (values[j] > values[j + 1]) {
            tmp = values[j];
            values[j] = values[j + 1];
            values[j + 1] = tmp;
        }
    }
    cout << values[0] << endl;
    cout << values[1] << endl;
    cout << values[2] << endl;
    cout << values[3] << endl;
    return 0;
}
