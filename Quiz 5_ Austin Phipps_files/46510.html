This program returns the index of the maximum value in the array. When values are equal, the left index is chosen due to the ">=" operator.

public class Main {
    public static int myFun(int[] arr, int left, int right) {
        if (left == right) {
            return left;
        }
        int mid = (left + right) / 2;
        int n1 = myFun(arr, left, mid);
        int n2 = myFun(arr, mid + 1, right);
        if (arr[n1] >= arr[n2]) {
            return n1;
        } else {
            return n2;
        }
    }

    public static void main(String[] args) {
        int[] arr = {5, 7, 2, 7};
        System.out.println(myFun(arr, 0, arr.length - 1));
    }
}
